# 资源管理

## 概述

`iFlutter` 提供了完整的 Flutter 资源管理解决方案，覆盖从资源注册、分类整理到质量检查的全流程，大幅减少手动操作 `pubspec.yaml` 的工作量。

## 功能列表

| 功能 | 说明 |
|------|------|
| [注册资源](part-1.md) | 自动感知文件变化，自动注册并生成类型安全的 `_res.dart` |
| [注册资源（目录方式）](part-2.md) | 以目录为单位注册，简化 `pubspec.yaml` |
| [注册字体](part-3.md) | 自动识别字体文件，完成 `pubspec.yaml` 中的字体配置 |
| [重置资源](part-4.md) | 重新扫描并生成资源文件，用于解决合并冲突或配置不同步问题 |
| [归纳资源](part-6.md) | 按模块对平展的资源文件进行自动分类整理 |
| [检查重复资源](part-5.md) | 检测项目中内容相同的重复资源文件 |
| [检查无用资源](part-7.md) | 扫描已注册但未被代码引用的冗余资源 |
| [监控图片资源](part-8.md) | 添加图片时实时检测大小，集成 TinyPNG 压缩 |
| [管理图片资源](part-9.md) | 可视化图片资源管理器，支持搜索、压缩、排序、删除 |
