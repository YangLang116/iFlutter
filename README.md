## 插件安装

- 插件安装: 插件已上传官方Plugins仓库，可搜索 `iFlutter` 下载
- 插件文档: [点击跳转](https://iflutter.toolu.cn)
- 插件改为最低兼容211版本，如遇到无法安装，请升级AndroidStudio
