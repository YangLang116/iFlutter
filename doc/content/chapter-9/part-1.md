要在 IntelliJ IDEA 切换JBR，请双击 Shift 键并选择 "Action"。在弹出的输入框中输入 "Choose Boot Java Runtime for the IDE"（输入 "Choose Boot" 也可以），然后在弹出的窗口中选择一个名为 "xxx JetBrains Runtime With JCEF" 的选项。单击 "OK" 按钮，等待下载完成并重启 IntelliJ IDEA 即可。
