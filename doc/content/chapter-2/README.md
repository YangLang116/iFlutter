# 代码生成

## 概述

`iFlutter` 提供了一系列代码生成工具，帮助开发者减少重复性编码工作，专注于业务逻辑实现。

## 功能列表

| 功能 | 说明 |
|------|------|
| [JSON 转 Dart Entity](part-1.md) | 将 JSON 数据一键转换为带空安全支持的 Dart 实体类 |
| [生成 fromJson、toJson 方法](part-2.md) | 为已有实体类自动补全 JSON 序列化/反序列化方法 |
| [提供 Live Template](part-3.md) | 内置常用 Flutter 组件的代码模板，缩写 + Tab 即可展开 |
