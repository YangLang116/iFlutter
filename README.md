## iFlutter是一款辅助Flutter开发的 IDEA 插件

## 说明
- 插件安装：插件已上传官方Plugins仓库，可搜索 `iFlutter` 下载
- 功能说明：[Wiki](https://github.com/YangLang116/iFlutter/wiki)
  - 1、资源文件管理
  - 2、依赖树生成
  - 3、Dart代码生成
  - 4、Http接口Mock
  - 5、包检查更新
