# 依赖管理

## 概述

`iFlutter` 提供了一套完整的 Pub 依赖管理工具，从搜索、定位、提取到版本检查，覆盖依赖管理的全生命周期，让开发者无需频繁切换至浏览器或命令行。

## 功能列表

| 功能 | 说明 |
|------|------|
| [搜索 Pub](part-1.md) | 在 IDE 内直接搜索和浏览 pub.dev 上的插件包 |
| [定位依赖](part-2.md) | 快速跳转到指定依赖的本地源码目录 |
| [提取依赖](part-3.md) | 一键将远程依赖转换为本地依赖，方便二次开发 |
| [检查更新](part-4.md) | 定时检测依赖版本，在 `pubspec.yaml` 中高亮显示可升级包 |
| [生成依赖树](part-5.md) | 可视化分析项目的完整包依赖关系 |
