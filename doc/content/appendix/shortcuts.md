# 快捷键汇总

以下是 `iFlutter` 插件涉及的所有快捷键，方便统一查阅。

## ⌨️ 全局快捷键

| 功能 | macOS | Windows / Linux |
|------|-------|----------------|
| 搜索 Pub 插件 | `Option + F` | `Alt + F` |
| 添加国际化资源 | `Option + A` | `Alt + A` |
| 移除国际化资源 | `Option + R` | `Alt + R` |

## 📝 编辑器快捷键

| 功能 | macOS | Windows / Linux | 说明 |
|------|-------|----------------|------|
| 触发快速修复 | `Option + Enter` | `Alt + Enter` | 在 `pubspec.yaml` 依赖版本处触发 `Update Package` |
| 展开 Live Template | `Tab` | `Tab` | 输入缩写后按 Tab 展开代码模板 |

## 🖼️ 资源管理器快捷键

| 功能 | macOS | Windows / Linux |
|------|-------|----------------|
| 搜索图片资源 | `Command + F` | `Ctrl + F` |
| 退出搜索模式 | `Escape` | `Escape` |

## 🌳 依赖树快捷键

| 功能 | macOS | Windows / Linux |
|------|-------|----------------|
| 搜索依赖结果 | `Command + F` | `Ctrl + F` |

## 🖼️ 媒体预览快捷键（MediaFilePreviewer）

| 功能 | macOS | Windows / Linux |
|------|-------|----------------|
| 缩放预览 | `Command + 滚轮` | `Ctrl + 滚轮` |
